Providing CSS

  • Embedded Stylesheet
  • External Stylesheet
  • Inline Styles

Embedded Stylesheet: It is a stylesheet which is embedded within the HTML file in head using <style> section.

External Stylesheet: It is a stylesheet used for CSS styling which is an external file. U need to link the external Stylesheet with the HTML file to style .
Example: <link rel="stylesheet" href="style.css">

Inline Styles: It is CSS style which is given with the line of HTML.
Example: <p> style="color=red;"

Basic selectors:

Using selectors we can select bunch of HTML code and style it using CSS.

  • Class:
      • Example: type.classname{......} or .classname{......}
        * We can have multiple elements with same class. All the elements in same class will be taking same styling if u choose .classname{......} method of selector
  • Id:
      • Example: #id{.......}
        * We can't have multiple elements with same Id . Id must be unique:)
  • Type:
      • Example: type{......}
        * All the elements in the type irrespective of class will be selected if we use type{....} selector method.
  • Attribute:
      • Its better if u dont use Attributes selectors. They are too fragile. But your this close to knowing it so here is an Example.
      Example: a[href^=https"]{....anchors starting with https....} or a[href*=google"]{....anchors containing google term in it....}

Pseudo Class

Pseudo Class are classes which are applied by browers default. All Pseudo Classes starts with colon. These are fragile too.

Example: article p: first-child{......}

  • article: last-child{......}
  • article h1: first-child{......}
  • article p: nth-child (odd){......}
For changing the links color which we visited and all anchors links color which we havent visited
    a:visited{ color:"..."} , a:link {color:"..."}
    Pseudo Classes
For changing the color of hover effect on any element
    p:hover , p:focus { color:"..."}

Pseudo element

Pseudo element are applied by browers default which is used to edit to style elements in particular style. All Pseudo elements starts with 2 colons.

Example: p::first-child {......}

  • p::first-letter {......}
  • p::first-line {......}

We can change the selection of text color using pseudo element

  • p::selection { background-color: pink;}

To make any selection as a single color

  • ::selection { background-color: pink;}
To insert any content before or after element
  • p::before{ content:"......"} i.e this will insert content before p element
  • p::after{ content:"......"} i.e this will insert content after p element
Block level element takes whole line. We can set display to block to make it block level element.
  • p::before{ content="..."; display = block;}
